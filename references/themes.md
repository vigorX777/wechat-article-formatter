# 微信文章排版主题

本文档定义了排版 Skill 支持的视觉主题。

## 1. Claude 风格（橙色主题 - 默认）

这是模仿 Claude 官方博客及 UI 风格的简约橙色主题，适合技术分享、深度思考类文章。

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#D97757` | Claude 橙，用于标题左边框、强调文字、引用块边框 |
| 主色浅 | `#E8956D` | 用于渐变或辅助装饰 |
| 背景浅 | `#FFF5F0` | 用于图片占位符背景、高亮背景 |
| 背景灰 | `#FAF9F7` | 用于表格交替行、代码块背景、引用块背景 |
| 文字色 | `#2D2D2D` | 正文文字颜色 |
| 标题色 | `#1A1A1A` | 标题文字颜色 |

### 核心元素样式：
- **二级标题 (h2)**: 12px 左边框 (#D97757)，20px 字号，加粗。
- **引用块 (blockquote)**: 4px 左边框 (#D97757)，#FAF9F7 背景，#666 文字。
- **列表项强调**: 强调文字使用主色加粗。

## 2. 自定义主题

用户可以通过指定色值来自定义主题。Skill 应能自动计算辅助色：
- **辅助色**: 主色降低饱和度或提高亮度。
- **背景浅色**: 主色极高亮度（如 95% 亮度）。
