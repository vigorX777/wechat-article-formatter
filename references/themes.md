# 微信文章排版主题

本文档定义了排版 Skill 支持的视觉主题。触发 Skill 后，必须先询问用户选择主题。

---

## 1. Claude 风格（简约橙 - 默认）

这是模仿 Claude 官方博客及 UI 风格的简约橙色主题，经过排版精修（行高 1.75、字间距 0.05em），适合技术分享、深度思考类文章。

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#D97757` | Claude 橙，用于标题左边框、强调文字、引用块边框 |
| 主色浅 | `#E8956D` | 用于渐变或辅助装饰 |
| 背景浅 | `#FFF5F0` | 用于图片占位符背景、高亮背景 |
| 背景灰 | `#FAF9F7` | 用于表格交替行、代码块背景、引用块背景 |
| 文字色 | `#2D2D2D` | 正文文字颜色 |
| 标题色 | `#1A1A1A` | 标题文字颜色 |

### 核心元素样式
- **全局布局**: 行高 1.75，字间距 0.05em，极淡背景容器 (rgba(0,0,0,0.02))
- **二级标题 (h2)**: 4px 左边框 (#D97757)，底部 1px 虚线边框，20px 字号，加粗
- **引用块 (blockquote)**: 4px 左边框 (#D97757)，#FAF9F7 背景，#666 文字
- **强调文字**: 主色加粗 (#D97757)
- **分隔线 (hr)**: 渐变效果

### 适用场景
- 技术分享
- 日常随笔
- 简短内容
- 产品介绍

## 2. 橙韵风格（杂志卡片）

渐变头图 + 卡片布局的杂志风主题，适合深度解读、专题报道类长文。

### 配色方案

> 完整配色表详见 [chengyun-element-styles.md](chengyun-element-styles.md)。

### 核心元素样式

- **头部区域**: 橙色渐变背景 (135deg, #fb923c → #ea580c)，底部圆角 24px
- **胶囊标签**: 半透明白底 rgba(255,255,255,0.2)，圆角 20px
- **主标题**: 26px 白色加粗
- **白色卡片**: 负 margin 上移 -20px，圆角 16px，阴影 rgba(0,0,0,0.06)。**内容仅放一段精炼 hook（2-3 句），自我介绍和过渡内容放卡片之后**
- **章节标题**: 中文序号方块（一、二、三...），40x40px，渐变背景，圆角 12px
- **引用块**: 浅黄渐变背景 (#fff9e6 → #fef3c7)，5px 左边框 (#d97706)
- **分隔线**: 渐变效果（两端透明，中间橙色 60%透明度）
- **正文**: 14px，行高 2.4，字间距 0.3px

### 页面结构

```
┌─────────────────────────────────────┐
│  头部渐变区域（橙色渐变）            │
│  - 标签（半透明白底胶囊）            │
│  - 主标题（白色大字）                │
│  - 副标题（半透明白字）              │
│  - 日期信息（半透明白字）            │
└─────────────────────────────────────┘
     ┌─────────────────────────┐
     │  白色内容卡片            │  ← 负 margin 上移
     │  - 开篇引言（仅1段hook） │
     └─────────────────────────┘
┌─────────────────────────────────────┐
│  正文区域                            │
│  - 分隔线                            │
│  - 旋转贴纸（01 PARALLEL）          │  ← 核心特色
│  - 章节标题                          │
│  - 正文段落                          │
│  - 引用块 / 提示块                   │
│  - ...重复...                        │
│  - 底部署名                          │
└─────────────────────────────────────┘
```

### 适用场景
- 教程指南
- 技术解析
- 使用技巧
- 趣味科普
- 步骤说明

### ⚠️ 兼容性注意
`transform: rotate()` 在部分微信客户端（特别是老旧 Android WebView）可能不生效，降级为水平显示。用户已知悉此风险。

## 4. 自定义色系功能

每个主题都有默认色系，但支持用户提供自定义主色来派生整套色系。

### 色系派生逻辑

当用户提供一个自定义主色（如 `#007AFF`）时，Claude 需要基于该主色自动派生出完整的色系：

#### 派生规则表

| 色系角色 | 派生方式 | 示例（主色 `#007AFF`） |
|:---|:---|:---|
| **主色** | 用户提供，直接使用 | `#007AFF` |
| **主色-深** | 主色降低亮度 15-20% | `#0062CC` |
| **主色-浅** | 主色提高亮度 10-15% | `#3399FF` |
| **强调色** | 主色微调饱和度或使用主色本身 | `#0066E6` |
| **背景-浅** | 主色亮度提高到 95%+，饱和度降低到 10-20% | `#EBF5FF` |
| **背景-渐变起点** | 主色亮度 90%+，饱和度 15-25% | `#E6F2FF` |
| **背景-渐变终点** | 主色亮度 85-90%，饱和度 20-30% | `#CCE5FF` |
| **引用块文字** | 主色降低亮度 40-50%，保持色相 | `#004080` |

#### 渐变色派生

头部渐变区域：
```
background: linear-gradient(135deg, {主色} 0%, {主色-深} 100%)
```

引用块背景渐变：
```
background: linear-gradient(135deg, {背景-渐变起点} 0%, {背景-渐变终点} 100%)
```

分隔线渐变（使用主色 60% 透明度）：
```
background: linear-gradient(90deg, transparent, rgba({主色RGB}, 0.6), transparent)
```

### 各主题的色系角色映射

| 主题 | 主色 | 主色-深 | 强调色 | 背景-浅 |
|:---|:---|:---|:---|:---|
| Claude 风格 | `#D97757` | `#C4684A` | `#D97757` | `#FFF5F0` |
| 橙韵风格 | `#fb923c` | `#ea580c` | `#d97706` | `#fff9e6` |
| 蓝色专业 | `#2563eb` | `#1d4ed8` | `#3b82f6` | `#eff6ff` |
| 贴纸风格 | `#D97757` | `#C4684A` | `#D97757` | `#FFF5F0` |
