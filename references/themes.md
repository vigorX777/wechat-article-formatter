# 微信文章排版主题

本文档定义了排版 Skill 支持的视觉主题。触发 Skill 后，必须先询问用户选择主题。

---

## 1. Claude 风格（简约橙 - 默认）

这是模仿 Claude 官方博客及 UI 风格的简约橙色主题，适合技术分享、深度思考类文章。

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#D97757` | Claude 橙，用于标题左边框、强调文字、引用块边框 |
| 主色浅 | `#E8956D` | 用于渐变或辅助装饰 |
| 背景浅 | `#FFF5F0` | 用于图片占位符背景、高亮背景 |
| 背景灰 | `#FAF9F7` | 用于表格交替行、代码块背景、引用块背景 |
| 文字色 | `#2D2D2D` | 正文文字颜色 |
| 标题色 | `#1A1A1A` | 标题文字颜色 |

### 核心元素样式
- **二级标题 (h2)**: 4px 左边框 (#D97757)，20px 字号，加粗
- **引用块 (blockquote)**: 4px 左边框 (#D97757)，#FAF9F7 背景，#666 文字
- **强调文字**: 主色加粗 (#D97757)

### 适用场景
- 技术分享
- 日常随笔
- 简短内容
- 产品介绍

## 2. 橙韵风格（杂志卡片）

渐变头图 + 卡片布局的杂志风主题，适合深度解读、专题报道类长文。

### 配色方案

#### 主色调（橙色系）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色-亮 | `#fb923c` | 渐变起始色 |
| 主色-深 | `#ea580c` | 渐变结束色 |
| 强调色 | `#d97706` | 文字高亮、左边框、序号方块 |
| 深橙 | `#78350f` | 引用块文字 |

#### 背景色

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#fcfbf8` | 浅米色/暖白 |
| 内容卡片 | `#ffffff` | 纯白 |
| 引用块背景-亮 | `#fff9e6` | 渐变起始 |
| 引用块背景-深 | `#fef3c7` | 渐变结束 |

#### 文字色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正文 | `#4a4a4a` | 深灰，护眼 |
| 标题 | `#1a1a1a` | 近黑 |
| 次要信息 | `#9ca3af` | 浅灰（日期、署名） |
| 白底高亮 | `#d97706` | 橙色强调 |

### 核心元素样式

- **头部区域**: 橙色渐变背景 (135deg, #fb923c → #ea580c)，底部圆角 24px
- **胶囊标签**: 半透明白底 rgba(255,255,255,0.2)，圆角 20px
- **主标题**: 26px 白色加粗
- **白色卡片**: 负 margin 上移 -20px，圆角 16px，阴影 rgba(0,0,0,0.06)
- **章节标题**: 中文序号方块（一、二、三...），40x40px，渐变背景，圆角 12px
- **引用块**: 浅黄渐变背景 (#fff9e6 → #fef3c7)，5px 左边框 (#d97706)
- **分隔线**: 渐变效果（两端透明，中间橙色 60%透明度）
- **正文**: 14px，行高 2.4，字间距 0.3px

### 页面结构

```
┌─────────────────────────────────────┐
│  头部渐变区域（橙色渐变）            │
│  - 标签（半透明白底胶囊）            │
│  - 主标题（白色大字）                │
│  - 副标题（半透明白字）              │
│  - 日期信息（半透明白字）            │
└─────────────────────────────────────┘
     ┌─────────────────────────┐
     │  白色内容卡片            │  ← 负 margin 上移
     │  - 开篇引言              │
     └─────────────────────────┘
┌─────────────────────────────────────┐
│  正文区域                            │
│  - 分隔线                            │
│  - 章节标题（序号方块 + 标题）       │
│  - 正文段落                          │
│  - 引用块                            │
│  - ...重复...                        │
│  - 底部署名                          │
└─────────────────────────────────────┘
```

### 序号对照表

| 序号 | 中文 |
|------|------|
| 1 | 一 |
| 2 | 二 |
| 3 | 三 |
| 4 | 四 |
| 5 | 五 |
| 6 | 六 |
| 7 | 七 |
| 8 | 八 |
| 9 | 九 |
| 10 | 十 |

### 适用场景
- 深度解读
- 专题报道
- 长文精读
- AMA/访谈整理
- 年度总结

## 3. 贴纸风格（旋转贴纸）

旋转贴纸编号 + 渐变头图 + 卡片布局的活泼风主题，适合教程指南、技术解析、趣味科普类文章。

### 配色方案

#### 主色调（Claude 橙色系）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#D97757` | Claude 橙，贴纸背景、强调色 |
| 主色深 | `#C4684A` | 渐变结束色 |
| 主色浅 | `#E8956D` | 渐变辅助 |

#### 背景色

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#FFFFFF` | 纯白 |
| 内容卡片 | `#ffffff` | 纯白 |
| 引用块背景 | `#FAF9F7` | Claude 灰 |
| 提示块背景 | `#FFF5F0` | 浅橙 |

#### 文字色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正文 | `#2D2D2D` | 深灰 |
| 标题 | `#1A1A1A` | 近黑 |
| 引用块 | `#666666` | 中灰 |
| 提示块 | `#8A6D3B` | 棕色 |
| 次要信息 | `#999999` | 浅灰（署名） |

### 核心元素样式

- **头部区域**: 橙色渐变背景 (135deg, #D97757 → #C4684A)，底部圆角 24px
- **胶囊标签**: 半透明白底 rgba(255,255,255,0.2)，圆角 20px，间距 8px
- **主标题**: 26px 白色加粗
- **白色卡片**: 负 margin 上移 -20px，圆角 16px，阴影 rgba(0,0,0,0.06)
- **旋转贴纸**: `transform: rotate(-15deg)`，两位数序号 + 英文标签
- **章节标题**: 22px 深色加粗
- **H3 小节标题**: 18px 橙色加粗
- **引用块**: #FAF9F7 背景，4px 左边框 (#D97757)，圆角 12px
- **提示块**: #FFF5F0 背景，4px 左边框 (#D97757)，圆角 12px
- **分隔线**: 渐变效果（两端透明，中间橙色 60%透明度）
- **正文**: 16px，行高 2.2，字间距 0.3px

### 贴纸英文标签对照表

| 场景 | 英文标签 |
|------|----------|
| 开篇/引言 | INTRO |
| 核心观点 | KEY POINT |
| 技巧/提示 | TIPS |
| 步骤 | STEP |
| 案例 | CASE |
| 对比 | COMPARE |
| 总结 | SUMMARY |
| 结论 | CONCLUSION |
| 注意事项 | NOTICE |
| 常见问题 | FAQ |
| 配置/设置 | CONFIG |
| 工具 | TOOLS |
| 最佳实践 | BEST PRACTICE |

### 页面结构

```
┌─────────────────────────────────────┐
│  头部渐变区域（橙色渐变）            │
│  - 标签（半透明白底胶囊）            │
│  - 主标题（白色大字）                │
│  - 副标题（半透明白字）              │
│  - 来源信息（半透明白字）            │
└─────────────────────────────────────┘
     ┌─────────────────────────┐
     │  白色内容卡片            │  ← 负 margin 上移
     │  - 开篇引言              │
     └─────────────────────────┘

│  正文区域                            │
│  - 分隔线                            │

│  - 章节标题                          │
│  - 正文段落                          │
│  - 引用块 / 提示块                   │
```

### 适用场景
- 教程指南
- 技术解析
- 使用技巧
- 趣味科普
- 步骤说明

### ⚠️ 兼容性注意
`transform: rotate()` 在部分微信客户端（特别是老旧 Android WebView）可能不生效，降级为水平显示。用户已知悉此风险。

## 4. 自定义主题（高级）

用户可以通过指定主色来自定义主题。Skill 应能自动计算辅助色：
- **辅助色**: 主色降低饱和度或提高亮度
- **背景浅色**: 主色极高亮度（如 95% 亮度）
