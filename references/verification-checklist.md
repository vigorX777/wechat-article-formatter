## 验证清单

> ⚠️ 本清单只覆盖 **HTML / 样式 / 版式** 验证，不覆盖 `publish.ts` 的发布态校验。标题是否持久化、摘要填写、保存成功信号、阻塞弹窗、草稿保存 verdict 等，需按 README / SKILL / 发布脚本的当前规则单独验证。

### 通用样式检查
- [ ] 所有样式是否已内联（无 `<style>` 标签用于内容区域）？
- [ ] 代码块的长行是否已处理为可自动换行的 HTML 结构？
- [ ] 图片占位说明文字是否优先使用源 Markdown 的 alt 文案 / 配图备注，而不是原始文件名？
- [ ] 复制功能是否使用 Clipboard API + text/html blob？
- [ ] 所有正文段落 font-size 是否为 16px？
- [ ] 所有 `<th>` 和 `<td>` 是否**显式**包含 `font-size: 16px`？
- [ ] HTML 内容容器是否使用 `id="output"`？
- [ ] 预览提示文字和复制按钮是否在 `#output` 和 `.preview-wrapper` **外部**（fixed 定位）？
- [ ] 封面图（WECHATIMGPH_1）是否在前言段落**之后**（而非之前）？
- [ ] 白色内容卡片是否**只包含 1-2 段精炼的 hook / 引言**？自我介绍和过渡内容是否在卡片之外？
- [ ] 头部标签、署名、主标题、副标题是否来自源 Markdown 明确定义字段，而不是在 HTML 阶段临时猜测或写死？
- [ ] 需要圆角的 `<table>`（如图片占位符）是否使用 `border-collapse: separate; border-spacing: 0`？（`border-collapse: collapse` 会导致 `border-radius` 失效）

### Claude 风格检查
- [ ] 表格背景色是否已正确应用到 `td` 标签？
- [ ] 引用块 (blockquote) 的边框是否符合主题配色？
- [ ] 段落行高为 1.75 (line-height: 1.75)？
- [ ] 字间距为 0.05em (letter-spacing: 0.05em)？
- [ ] H2 有底部虚线 (border-bottom: 1px dashed)？
- [ ] 分隔线使用渐变 (linear-gradient)？
- [ ] 外层容器有极淡背景 (rgba(0,0,0,0.02))？

### 橙韵风格检查
- [ ] 头部渐变区域是否完整（标签、主标题、副标题、署名；日期信息如有则额外检查）？
- [ ] 白色内容卡片是否使用负 margin 上移？
- [ ] 章节标题是否使用中文序号方块（一、二、三...）？
- [ ] 引用块是否使用浅黄渐变背景和左边框？
- [ ] 分隔线是否使用渐变效果？

### 蓝色专业风格检查
- [ ] 头部蓝色渐变区域是否完整（标签、主标题、副标题、署名；数据来源如有则额外检查）？
- [ ] 白色内容卡片是否使用负 margin 上移和圆角阴影？
- [ ] 章节标题是否使用蓝色中文序号方块（一、二、三...）？
- [ ] 引用块是否使用浅蓝渐变背景和蓝色左边框？
- [ ] 分隔线是否使用蓝色渐变效果？
- [ ] 强调文字是否使用 `#2563eb` 蓝色？

### 贴纸风格检查
- [ ] 头部橙色渐变区域是否完整（标签、主标题、副标题、署名；来源信息如有则额外检查）？
- [ ] 白色内容卡片是否使用负 margin 上移和圆角阴影？
- [ ] 旋转贴纸是否使用 `transform: rotate(-15deg)`？
- [ ] 贴纸序号是否使用两位数格式（01, 02...）+ 英文标签？
- [ ] 引用块是否使用 `#FAF9F7` 灰色背景和橙色左边框？
- [ ] 提示块是否使用 `#FFF5F0` 浅橙背景？
- [ ] 分隔线是否使用橙色渐变效果？
- [ ] 强调文字是否使用 `#D97757` 橙色？
